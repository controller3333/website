# 効果音ノードジェネレーター

ゲーム用SEをノードで組み立てるローカルブラウザツールです。

`index.html` をブラウザで直接開けば使えます。サーバーやビルドは不要です。

## できること

- ノードを配置して線で接続
- オシレーター、ノイズ、スイープ、LFO、エンベロープ、ゲイン、フィルター、
  歪み、ビット削り、ディレイ、リバーブ、ミキサー、リング変調、サンプル、
  出力ノード
- 木の足音、鍵束、木片落下、裏口の物音、心核共鳴の初期プリセット
- 再生、停止、生成、波形確認、WAV書き出し
- プロジェクトJSONの保存と読み込み

## 操作

- 背景を左ドラッグ: ノード画面を移動
- マウスホイール/トラックパッド: ノード画面をスクロール
- Ctrl/Command + ホイール: ズーム
- ノードのヘッダーを左ドラッグ: ノードを移動

## メモ

WAV書き出しはブラウザ内で行います。OGG変換は確認後にFFmpegで行う想定です。
