# プロシージャルSEジェネレーター

ノードベースのプロシージャル効果音合成ツールです。外部サンプル素材は読み込まず、オシレーター、ノイズ、変調、フィルター、空間系エフェクトなどの内部生成だけでワンショットSEを作ります。

ブラウザだけで動く、依存ゼロのバニラJSアプリです。各ノードはオフラインでサンプルバッファを処理し、グラフ全体を評価して1本の音をレンダリングします。再生、WAV書き出し、波形表示に対応しています。

## 起動

ローカルの静的サーバで `index.html` を開きます。音声まわりで `file://` だと制限が出る場合があるため http 推奨です。

```sh
cd ProceduralSEGenerator
python -m http.server 5599
# http://localhost:5599
```

`npx --yes http-server . -p 5599 -c-1` でも起動できます。

## 使い方

- 左パレットのノード名をクリックするとキャンバス中央に追加されます。
- ノードの出力ポートから入力ポートへドラッグして接続します。配線クリックで削除できます。
- ノードのヘッダをドラッグで移動できます。`×` で削除できます。出力ノードは固定です。
- 選択中ノードは `Delete` キーで削除できます。
- 選択ノードは `Ctrl+C` / `Ctrl+V` でコピー&ペースト、`Ctrl+D` で複製できます。
- ノードにカーソルを重ねて `Ctrl+Space` を押すと、その地点までの音を試聴できます。
- 背景ドラッグまたは右クリックドラッグでパン、ホイールでズームします。
- 上部バーの `再生`、`停止`、`生成`、`WAV` で再生・停止・再レンダリング・書き出しを行います。
- `ピッチ±` とサイコロボタンで、再生ごとにピッチとシードを少し揺らせます。足音や着弾音の反復感を減らす用途に向いています。
- 最終的に音が出るのは、出力ノードに繋がった信号だけです。

## ノード一覧

| カテゴリ | ノード | 役割 |
|---|---|---|
| 音源 | オシレーター | サイン、矩形、ノコギリ、三角波。FM/PM入力、ユニゾン、デチューンに対応 |
| 音源 | ノイズ | ホワイト、ピンク、ブラウンノイズ。シード指定で再現可能 |
| 変調 | LFO | 低周波オシレーター。深さと中心値で制御信号を作成 |
| 変調 | エンベロープ(ADSR) | 立上り、減衰、保持、余韻、ゲート。入力に乗算、未接続なら包絡線そのものを出力 |
| 変調 | スイープ | 時間で開始値から終了値へ変化する制御値。ピッチやフィルター掃引に使用 |
| 変調 | サンプル&ホールド | 入力または内部ランダム値を周期的にホールド。階段状のピッチ/フィルター変調に使用 |
| 加工 | ゲイン/アンプ | ゲイン調整。CV入力で振幅変調 |
| 加工 | フィルター | ローパス、ハイパス、バンドパス。カットCV入力でカットオフ変調 |
| 加工 | 歪み | tanh、クリップ、折り返し歪み |
| 加工 | ビットクラッシュ | ビット深度とサンプル間引きでレトロ/デジタル感を付与 |
| 加工 | タイムシフト | レイヤー全体を指定秒数だけ後ろにずらす |
| 加工 | ディレイ/エコー | ディレイとフィードバック |
| 加工 | リバーブ | 簡易Schroederリバーブ |
| 合成 | ミキサー | 最大4入力をレベル付きで合成。ステレオ信号も扱います |
| 合成 | リング変調 | 2信号の乗算。金属的・ベル系の音作りに使用 |
| ステレオ | パンナー | モノからステレオへ変換。パンCVとHaas風の広がりに対応 |
| 出力 | 出力 | マスターゲイン、正規化、クリップ。グラフの最終出力 |

## プリセット

レーザーショット、爆発、コイン/取得、パワーアップアルペジオ、打撃/インパクト、足音、UIブリップ、風/ウーッシュ、ステレオ・ウーッシュ、ロボットビープ(S&H)、基本形を収録しています。

## 構成

```text
index.html
css/style.css
js/
  dsp.js       DSPエンジンと全ノード定義
  graph.js     グラフモデルとオフライン評価
  audio.js     再生、WAV書き出し、波形描画
  editor.js    ノードエディタUI
  presets.js   プリセット
  main.js      起動、パレット、保存/読込
```

## メモ

- 接続は音声信号も制御信号も同じサンプルバッファで扱います。
- レンダリングは出力ノードから入力を再帰的に評価します。
- ステレオ非対応ノードにステレオ信号が入った場合は、自動でモノへダウンミックスします。
